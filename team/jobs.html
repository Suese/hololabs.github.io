1. Full-stack Game/Web Developer

We are currently expanding our team, looking for programmers who have web and/or game development experience. The requirements listed below are broad and multidisciplinary. We don't expect candidates to necessarily have all these skills, but as an indie studio we are looking for someone who can "wear many hats". Ultimately, we seek passionate and motivated team players who will grow with the studio and help to create amazing new products in the future.

Required skills:
Highly organized, self-motivated, with strong problem solving, troubleshooting, and analytical skills
Can self-manage, work in an agile environment (e.g., SCRUM), and track progress
Familiarity with version control (Git)
Game development:
Intimate knowledge of Unity3D
Interested in exploring the new WebGL publishing feature in Unity 5
Highly competent in C#: should understand coroutines, events, delegates, singletons, etc.
Experience in game programming concepts: 3D math, colliders, physics, character controllers, GUI, cameras
Experience with some popular Unity3D packages: NGUI, Playmaker, TouchScript, AStar (path planning)
Web development:
Experience developing cloud-based applications, working with web APIs, parsing JSON, data persistence, and database/CMS interaction
Knowledge of web application programming (HTML5, responsive UI, AngularJS, Bootstrap, Javascript, jQuery, etc.)
Interested in 3D web applications (WebGL)
Other nice-to-have skills:
Design experience (e.g., Photoshop, 3DSMax, Maya, etc.)
Experience writing shaders
Animation (Mecanim, animation controllers, rigging and retargeting)
Mobile app development (iOS/Android): XCode, C++, NDK, Obj-C
Bilingual (English/French)
Miscellany:
Looking for full-time candidates in Montreal, Canada
Remote working is NOT possible for this position
Compensation according to experience (contract or salary; your choice)
